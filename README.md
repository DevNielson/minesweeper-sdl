## Pré-requisitos

* [SDL 3.4.0](https://github.com/libsdl-org/SDL) ou superior.
* C++ 23 ou superior.
* [Make](https://www.gnu.org/software/make/#download).

## Instruções para compilar

### Windows om MinGW ou MSYS2

1. Renomei o seu arquivo SDL para SDL3
2. Mova/Copie a pasta SDL3 para minesweeper-sdl\
3. Abra o cmd ou PowerShell
4. Navegue até a pasta minesweeper-sdl\build\
5. No terminal execute o comando make
6. E copie o arquivo SDL3\bin\SDL3.dll para a pasta minesweeper-sdl\bin\
